This uses MEAN framework.
MEAN - MongoDB, ExpressJS, AngularJS, NodeJS. based fullstack js framework

To debug/modify the app locally use node.js
1.) cd to the project directory
2.) Enter 'npm install' in the node console to load all the dependecies.
3.) Enter 'npm start' to run the app in local browser

Since this application is running on server by Heroku.
Follow these steps to deploy any changes:

pre-requisite : Download https://toolbelt.heroku.com/
